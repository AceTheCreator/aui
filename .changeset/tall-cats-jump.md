Add Web Component support: `<aui-asyncapi>` and `<aui-asyncapi-renderer>` custom elements, available via the `apiuikit/webcomponents` subpath, so apiuikit can be used from any framework or plain HTML without React. See [docs/usage/with-webcomponents.md](../docs/usage/with-webcomponents.md).
